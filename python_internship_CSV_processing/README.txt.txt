Python script responsible for reading and formating CSV files.

Script might not generate correct data if loaded CSV file will contain correct values in incorrect 'column'

To run file:
1) Open CMD and navigate to directory where main.py is storred.
2) Run the script using [script_name] [CSV file]
3) Example - If the CSV name is test.csv and the name of the script remained unchanged run it: main.py test.csv

Program created with Python 3.6.1
